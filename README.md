# Gambits.in

**Victory favors the bold.**

Gambits.in is a high-performance, conversion-focused landing page for an upcoming chess education platform dedicated to the art of the sacrifice. This project serves as the initial pre-launch waitlist to gather interest for the "ultimate interactive library for chess gambits."

## About the Project

This repository contains the source code for the [Gambits.in](https://gambits.in) landing page. It is built with a modern tech stack focusing on minimalism, performance, and a dark, mysterious aesthetic fitting the tactical depth of chess.

**Current Status:** Pre-launch / Development

The vision is to build a startup within the chess domain, offering tools and training specifically focused on gambit lines and tactical sacrifices.

## Technology Stack

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animation:** HTML5 Canvas (Custom interactive background)
- **Deployment:** Vercel (Static Export compatible)
- **Language:** TypeScript

## Features

- **Interactive Visuals:** A custom-built, mouse-reactive "neural network" background simulating tactical calculations.
- **Performance:** Highly optimized static build with minimal bundle size.
- **Email Capture:** Integrated form for waitlist management (ready for Formspree/backend integration).
- **Responsive Design:** Mobile-first approach ensuring perfect rendering across all devices.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/r-baruah/gambits.in.git
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
